# JOGO-ALGORITMO-C
Jogo desenvolvido para fins educacionais voltado para o aprendizado da linguagem C
# 🎮 Mini Games em C

Projeto acadêmico desenvolvido em linguagem C, com execução via terminal, contendo um sistema de minijogos interativos. O programa apresenta um menu principal com três opções de jogos e permanece em execução até o usuário escolher sair.

# 📚 Sobre o projeto

Este projeto foi criado com o objetivo de aplicar, na prática, conceitos fundamentais de algoritmos e lógica de programação por meio da construção de um arcade com três minijogos em console. Entre os objetivos da atividade estão a resolução de problemas, o raciocínio lógico e a aplicação prática dos conteúdos estudados.




#🕹️ Minijogos


#1. Pergunta e Resposta
Jogo de perguntas de múltipla escolha. O jogador responde questões com quatro alternativas, e o sistema informa se a resposta está correta ou incorreta. Ao final, é exibido o desempenho do jogador. No código atual, foram implementadas cinco perguntas fixas. 


#2. Cobra na caixa
Jogo para dois jogadores. Existem cinco caixas: uma contém o botão que garante a vitória e outra esconde uma cobra, causando a derrota. A posição dos elementos é sorteada aleatoriamente, e os jogadores se alternam tentando encontrar o botão sem cair na cobra. 






#3. goumas war
Jogo de estratégia em que dois jogadores controlam Gousmas, criaturas que acumulam fúria ao serem atacadas. Cada jogador começa com duas Gousmas, podendo atacar ou dividir a fúria entre elas. Caso o nível de fúria ultrapasse o limite, a Gousma se desintegra. Vence quem eliminar todas as Gousmas do adversário. 


# Funcionalidades

- Menu principal com navegação entre os minijogos
- Interação via terminal
- Sistema de perguntas com múltiplas alternativas
- Sorteio aleatório no jogo Cobra na Caixa
- Estrutura com `struct` no jogo Gousmas War
- Opção de reiniciar o minijogo ou voltar ao menu principal
